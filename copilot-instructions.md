# Responsible Generative AI Customer Service Framework

Architecture

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

Backend:
- Django
- Django REST Framework

Database:
- PostgreSQL
- pgvector

AI:
- LangChain
- OpenAI GPT-4.1

Requirements

- Modular architecture
- SOLID principles
- UUID primary keys
- JWT authentication
- Service layer
- Repository pattern where appropriate
- Type hints
- PEP8
- Comprehensive docstrings

Every module should be independent.

Every API endpoint should be documented.

Every database model should include timestamps.

Every response should be logged.

Follow enterprise software engineering best practices.
